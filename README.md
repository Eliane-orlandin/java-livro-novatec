# Java - Guia do Programador 📘

Este repositório contém os exercícios e exemplos que estou desenvolvendo durante meus estudos do livro **"Java - Guia do Programador"** da editora Novatec.  
O objetivo é praticar os conceitos apresentados no livro e consolidar meu aprendizado em Java.

---

## 📚 Objetivo
- Registrar minha evolução nos estudos de Java.
- Praticar programação orientada a objetos.
- Explorar recursos da linguagem e boas práticas.
- Criar uma base de código que possa servir de referência futura.

---

## 🛠️ Tecnologias utilizadas
- **Java** (versão recomendada pelo livro ou mais recente)
- **VS Code** como IDE principal
- **Git/GitHub** para versionamento e controle dos exercícios
